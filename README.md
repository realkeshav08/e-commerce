# Forever — Logo Assets

## Purpose
Provide a consistent, versatile brand identity for Forever across web, mobile, and print.

## Structure
- logos/
  - logo-full.svg — primary logo (mark + wordmark)
  - logo-icon.svg — standalone mark/icon
  - logo-horizontal.svg — horizontal lockup
  - logo-vertical.svg — stacked lockup
  - logo-favicon.ico — favicon
- colors/
  - brandColors.json — hex/rgba brand palette
- guidelines/
  - logo-guidelines.md — spacing, clearspace, min sizes, color usage

## Design Concepts
- Continuous thread forming an "F" or clothing outline
- Minimal single-line mark suggesting fabric or stitch
- Geometric, modern pattern with subtle gradient option
- Elegant spool motif for premium positioning

## File Formats & Exports
- Source: SVG (primary, editable)
- Raster: PNG at 48, 96, 192, 512 px
- Favicon: ICO and PNG (16×16, 32×32)
- Print: PDF/EPS if required

## Sizes & Clearspace
- Favicon: 16×16, 32×32
- App/icon: 48–512 px raster sizes
- Minimum logo display width: defined in logo-guidelines.md
- Clearspace: 1× icon height on all sides

## Color & Variants
- Primary, secondary, and neutral palettes in brandColors.json
- Light/dark reversed variants and single-color monotone versions

## Usage Notes
- Use SVG on web for sharp scaling
- Use PNG where SVG unsupported
- Do not alter proportions, stroke weights, or color values outside the defined palette
- Provide visual examples of approved and disallowed treatments in guidelines
